# Code Review Assistant

This skill helps review code for best practices, potential bugs, and suggests improvements.

## When to Use

Use this skill when:
- You want to ensure code follows best practices
- You need to identify potential bugs or issues
- You want suggestions for code improvements
- You're preparing code for production

## What It Does

- Analyzes code for common issues and bugs
- Checks adherence to coding standards and best practices
- Suggests improvements for readability, performance, and maintainability
- Provides actionable feedback

## How to Use

1. Share the code you want reviewed
2. Specify any particular concerns or focus areas
3. The assistant will provide detailed feedback
4. Apply suggested improvements as needed

## Best Practices

- Review code in small, manageable chunks
- Focus on one aspect at a time (style, logic, performance)
- Consider the context and requirements of the code
- Provide specific, actionable feedback
- Balance criticism with positive reinforcement
